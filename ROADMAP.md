# Roadmap

这个项目的目标是成为一个本地优先的微信聊天关系分析器，而不是商业化服务。下面是仍然值得继续迭代的方向。

## 已完成的产品模块

当前已完成 17 个模块。

- 普通用户桌面启动器
- 本地 Web UI
- 联系人列表与联系人搜索
- 聊天记录浏览
- 回忆搜索
- 关系报告
- 深度社交关系解读
- Markdown 关系报告导出
- 总览与年度报告
- 隐私安全中心
- 快照新鲜度提示
- 首次使用引导
- 快照刷新体验
- 媒体消息解析
- Demo 数据与开源展示
- 本地 AI 问答
- 安全增强（一键清理）

## 下一阶段优先级

核心功能已全部完成。以下是可继续迭代的方向，但不属于发布阻断项。

### 可选：进一步安全增强

- 密钥加密保存（本地 passphrase 加密 wechat_keys.json）
- 导出前二次确认弹窗
- 敏感字段打码（联系人列表中的用户名）
- 发布前自动检查工具（pre-commit hook）

## 不建议优先做

- 云端账号系统
- 上传聊天记录做分析
- 商业化 SaaS
- 自动绕过系统安全限制
- 面向他人设备的数据采集
